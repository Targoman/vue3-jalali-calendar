# vue3-jalali-calendar

## Installation

```sh
npm i vue3-jalali-calendar
```

## Sample Event:

```sh
	{
		id: 1,
		startDateTime: moment("2023-03-06T13:37:41.020+00:00"),
		endDateTime: moment("2023-03-06T20:49:41.020+00:00"),
		title: "رویداد شماره ۱",
		color: "#29B6F6",
	},
```

## Sample Vacation:
